# Menu Photos

Place your menu item photos in this folder with the following naming convention:

**Format:** `{item_id}_image.{extension}`

**Examples:**
- `coca_cola_image.jpg` (or .png, .jpeg, .webp)
- `vodka_image.jpg`
- `borovicka_image.jpg`
- `slivovica_image.jpg`
- `small_beer_image.jpg`
- `aperol_spritz_image.jpg`
- `wine_image.jpg`
- `sprite_image.jpg`
- `fanta_image.jpg`
- `orange_juice_image.jpg`
- `apple_juice_image.jpg`
- `chips_image.jpg`
- `bread_sticks_image.jpg`

**Supported formats:** .jpg, .jpeg, .png, .webp

**Recommended size:** 400x300px or similar aspect ratio for best display quality.
